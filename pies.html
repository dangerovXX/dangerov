пироги

Ватрушка

круглая, открытая сверху лепёшка с творогом в качестве наполнителя.

Кулебяка

пирог со сложной начинкой из мяса, рыбы, грибов, ягод или капусты, разделённой блинами.

Шаньга

круглые открытые пирожки из дрожжевого теста с разными начинками.

Калитка

небольшие открытые пирожки из пресного теста с различными начинками.

Расстегай

традиционная русская выпечка с открытой начинкой в середине. Начинкой может быть мясо, грибы или рис.

Пирог с визигой

небольшие открытые пирожки из пресного теста с различными начинками.

Расстегай с грибами

небольшие открытые пирожки из пресного теста с различными начинками.

Яблочный

это одна из самых распространённых разновидностей пирога, для начинки которого используются яблоки.
